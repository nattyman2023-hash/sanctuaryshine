# Sanctuary Shine - Cleaning Services Website

A production-ready static website for Sanctuary Shine, a domestic and commercial cleaning company serving Salford, Manchester, and all Greater Manchester boroughs.

Built with [Astro](https://astro.build) + [Tailwind CSS](https://tailwindcss.com).

## 🚀 Quick Start

### Prerequisites
- Node.js 18+ (recommended: 20+)
- npm 9+

### Install Dependencies
```bash
npm install
```

### Run Locally
```bash
npm run dev
```
The site will be available at `http://localhost:4321`.

### Build for Production
```bash
npm run build
```
The built files will be in the `dist/` folder.

### Preview Production Build
```bash
npm run preview
```

## 🚢 Deploying to Hostinger

### Step 1: Build the site
```bash
npm run build
```

### Step 2: Upload to Hostinger
1. Log into your Hostinger cPanel or hPanel
2. Go to **File Manager** → `public_html/`
3. Upload the entire contents of the `dist/` folder to `public_html/`
4. Also upload `public/send.php` to `public_html/` (for form handling)

### Step 3: Connect Domain
1. In Hostinger, go to **Domains** → point your domain to the hosting
2. Update the `site` URL in `astro.config.mjs` if your domain is different
3. Rebuild and re-upload if you change the domain

### Step 4: Configure Contact Form
The site includes a PHP form handler at `/send.php`. To make it work:

1. **Update the email address** in `public/send.php`:
   ```php
   $to_email = "your-email@domain.com"; // Change this
   ```
2. Upload `send.php` to your Hostinger server
3. Update the form `action` attributes in the Astro pages to point to `/send.php`

**Alternative form backends:**
- **Formspree:** Replace form action with `https://formspree.io/f/YOUR_FORM_ID`
- **EmailJS:** Use their JavaScript library (instructions on their website)
- **Google Forms:** Use Google Forms action URL

## 🎨 Brand Customization

### Colors
Edit `src/styles/global.css` to update brand colors:
- `--color-primary`: Aqua (#00C2D1)
- `--color-secondary`: Yellow (#FFD200)
- `--color-black`: #0B0B0B
- `--color-charcoal`: #1A1A1A

### Business Info
Edit `src/data/site.js` to update:
- Business name, phone, email, address
- Social media links
- Business hours

### Services
Edit `src/data/services.js` to add, remove, or modify services.

### Locations
Edit `src/data/locations.js` to update location pages and SEO content.

## 📝 Updating Content

### Adding a new page
1. Create a new `.astro` file in `src/pages/`
2. Use the `BaseLayout` component
3. Rebuild the site

### Updating page content
- Most page content is inline in the `.astro` files
- Data-driven content (services, locations) is in `src/data/`
- Edit the relevant file and rebuild

## 🔍 SEO Notes

- Each page has unique title and meta description
- Schema markup (LocalBusiness, Service, FAQ) is included
- Sitemap is auto-generated by Astro
- `robots.txt` is in the `public/` folder
- Open Graph tags are included for social sharing
- Canonical URLs are set for all pages

## 📦 What to Upload to Hostinger

Upload the **entire contents** of the `dist/` folder to your Hostinger `public_html/` directory. The `dist/` folder contains:

- `index.html` (homepage)
- `about/index.html`
- `services/index.html`
- `services/domestic-cleaning/index.html` (and other services)
- `areas/index.html`
- `areas/cleaners-salford/index.html` (and other locations)
- `contact/index.html`
- `get-a-quote/index.html`
- `privacy-policy/index.html`
- `terms-and-conditions/index.html`
- `favicon.svg`
- `robots.txt`
- `_astro/` (compiled CSS/JS assets)

Also upload `public/send.php` separately to `public_html/send.php`.

## 🧹 Maintenance

### To update the site:
1. Make changes to the source files
2. Run `npm run build`
3. Upload the new `dist/` contents to Hostinger

### To add a new location page:
1. Add the location data to `src/data/locations.js`
2. Rebuild — the page will be auto-generated

### To add a new service:
1. Add the service data to `src/data/services.js`
2. Rebuild — the service page will be auto-generated
